Timetable.js is a vanilla javascript plugin for creating great responsive timetables in a blink of an eye. It provides a simple javascript interface to add events and locations. Works like a charm on desktop and mobile. You can easily change the looks with css/sass.

Easy to use

Make a timetable object, optionally set the scope in hours (the visible hours in the timetable)

Changing the looks

Instead of adding the included css directly to your HTML, you could import the sass file. All colors and spacing values are defined as default variables which you can easily override.
